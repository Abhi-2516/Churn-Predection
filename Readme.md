# Customer Churn Prediction Project

This project predicts customer churn for a telecom company using a Random Forest model and SMOTE for handling class imbalance.

**Live App Demo:** [**<<https://churn-predection-o943k62vvgau8dondvvzwf.streamlit.app/ >>**]

## Project Overview
* **Objective:** To build a machine learning model that predicts customer churn.
* **Dataset:** The Telco Customer Churn dataset.
* **Model:** A Random Forest Classifier was trained, achieving 78% accuracy on the test set.
* **Technology:** Python, Pandas, Scikit-learn, SMOTE, and Streamlit.

## How to Run
1.  Clone this repository.
2.  Install the dependencies: `pip install -r requirements.txt`
3.  Run the Streamlit app: `streamlit run app.py`
